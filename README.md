# BookMyShow Clone 🎬

This is a frontend project inspired by the BookMyShow website.
The project is built using **HTML and CSS** and recreates a movie booking website interface.

## Features

* Homepage layout similar to BookMyShow
* Movie cards and banners
* Responsive UI design
* Navigation bar and sections for movies/events

## Technologies Used

* HTML
* CSS

## Project Structure

index.html
style.css

## Purpose of the Project

This project was created to p improve skills in building website.
